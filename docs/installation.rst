============
Installation
============


Stable release
--------------

To install Deepgroove WEB user experiment, run this command in your terminal:

.. code-block:: console

    $ pip install deepgroove_web_user_experiment

This is the preferred method to install Deepgroove WEB user experiment, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Deepgroove WEB user experiment can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/fosterrath-mila/deepgroove_web_user_experiment

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/fosterrath-mila/deepgroove_web_user_experiment/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/fosterrath-mila/deepgroove_web_user_experiment
.. _tarball: https://github.com/fosterrath-mila/deepgroove_web_user_experiment/tarball/master
